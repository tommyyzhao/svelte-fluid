# Contributing to svelte-fluid

The full contributing guide lives at [`docs/contributing.md`](docs/contributing.md).
It covers local setup, architecture rules, common workflows, and the
verification checklist.

**Quick start:**

```sh
bun install
bun run dev        # demo at localhost:5173
bun run test       # 126 tests
bun run check      # svelte-check (0 errors expected)
bun run prepack    # svelte-package + publint (must pass before publish)
```

Please open an issue before starting large changes so we can align on
direction.
