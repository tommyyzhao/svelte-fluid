# Session handoff — 2026-04-13

## What this session built

### Engine features

1. **roundedRect ContainerShape** — new shape type: fluid confined inside a rounded rectangle. Uses Inigo Quilez rounded-box SDF. Config: `{ type: 'roundedRect', cx, cy, halfW, halfH, cornerRadius }`.

2. **Frame cornerRadius** — the existing `frame` shape now accepts optional `cornerRadius`. When >0, the inner cutout uses a rounded-box SDF instead of sharp Chebyshev distance. Exposed via `<FrameFluid cornerRadius={0.06} />`.

3. **randomSplatSwirl** — new Bucket A config param. Per-splat tangential velocity relative to container/canvas center. `dx = -(y - cy) * swirl`, `dy = (x - cx) * swirl`. Positive = CCW, negative = CW.

4. **randomSplatEvenSpacing** — new Bucket A config param. When true, distributes splats evenly across the x-axis: `x = (i + 0.5) / count` instead of random.

5. **randomSplatSpread** — new Bucket A config param. Controls vertical jitter range (default 0.1 = +/-0.05). Set to 2.0 for full-canvas scatter with container shapes.

6. **Randomized burst timing** — interval between random splat bursts jittered +/-50% via `baseInterval * (0.5 + rng())` for organic feel.

7. **HDR area scaling** — `hdrMultiplier()` method scales the 10x HDR base by `sqrt(containerArea)` for each shape type, clamped at 3x. Prevents oversaturation in confined containers.

8. **Graceful WebGL context failure** — try-catch around `new FluidEngine()` in Fluid.svelte. Canvas CSS `background: #000` for clean black fallback.
